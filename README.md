# discuz-sdk
 
## 上传模块

将代码里的插件文件夹 `source/plugin/bapp` 拷贝到 *Discuz!* 下的 `source/plugin` 中

## 安装模块

找到「*Discuz!* 管理中心」> 「应用」> 「插件」> 「B.app支付」  

1. 点击 `安装`  
2. 点击 `启用`
3. 点击 `设置`，填上 B.app `App Key` `App Secret`
4. 点击 `提交`
